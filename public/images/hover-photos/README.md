# Hover Photos

Masukkan foto-foto Anda di folder ini untuk efek hover mouse!

## Format foto yang disarankan:
- Format: JPG, PNG, WEBP
- Ukuran: Persegi (misal 200x200px) agar terlihat bagus
- Nama file: foto1.jpg, foto2.png, foto3.jpg, dll.

## Foto yang akan ditampilkan:
- foto1.jpg - Foto pertama
- foto2.jpg - Foto kedua  
- foto3.jpg - Foto ketiga
- foto4.jpg - Foto keempat
- foto5.jpg - Foto kelima
- foto6.jpg - Foto keenam
- foto7.jpg - Foto ketujuh
- foto8.jpg - Foto kedelapan
- foto9.jpg - Foto kesembilan
- foto10.jpg - Foto kesepuluh

Jika ada lebih dari 10 foto, sistem akan secara otomatis mendeteksi dan menggunakannya.
