# Folder Images

Tempatkan foto-foto anniversary Anda di folder ini untuk digunakan dalam aplikasi scrollytelling.

## Format yang Didukung
- JPG/JPEG
- PNG
- WebP

## Penamaan File yang Disarankan
- `puzzle-photo.jpg` - Foto utama untuk puzzle (ukuran ideal: 400x400px)
- `special-memory.jpg` - Foto khusus untuk game bottle (foto yang dicari)
- `beginning.jpg` - Foto untuk section "The Beginning"
- `memories.jpg` - Foto untuk section "Beautiful Memories" 
- `milestones.jpg` - Foto untuk section "Milestones"
- `future.jpg` - Foto untuk section "Our Future"

## Tips
- Gunakan foto dengan resolusi yang baik (minimal 400x400px untuk puzzle)
- Foto berbentuk persegi cocok untuk puzzle
- Hindari foto yang terlalu gelap atau terlalu terang
- Kompres foto jika ukuran file terlalu besar (maksimal 2MB per foto)
- **special-memory.jpg** = foto yang akan ditemukan di dalam bottle game

## Cara Mengganti Foto
1. Masukkan foto ke folder ini
2. Ganti nama file sesuai kebutuhan
3. Update referensi di server.js jika perlu
4. Refresh browser untuk melihat perubahan
